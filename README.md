## Algorithm:
### Step 1: Import numpy module
### Step 2: prepare list from the each of the equations.

### Step 4: End the program.
## Program:
#Program to find the rank of a matrix.

import numpy as np

a=np.array([[1,2,3],[3,6,9]])

result=np.linalg.matrix_rank(a)

print(result)
